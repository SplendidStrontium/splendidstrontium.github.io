The Nanny likes to use the open-source Anki flashcard program to acquire new data-sets. The following are decks she has created.

Counting 1-20 (ENG)
Practice sight-reading numbers with young children. 119 cards, each with a spoken number. Visualizations are inspired by Common Core bars(5) and pips(1). Deck provides foundations for addition without using notation.
